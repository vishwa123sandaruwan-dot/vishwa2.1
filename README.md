# 🧬 vishwa 2.0 - WhatsApp Bot

vishwa 2.0 යනු vishwa sandaruwan විසින් වැඩිදියුණු කරන ලද, නවීන AI තාක්ෂණයෙන් යුත් සිංහල WhatsApp බොට් ලෙකී. 

---

### 🚀 බොට් එක පණ ගැන්වීමට පියවර (Setup Guide)

බොට් එක සාර්ථකව ක්‍රියාත්මක කිරීමට පහත පියවර අනුගමනය කරන්න:

#### 1. Session ID එක ලබා ගැනීම
බොට් එකට ඔබේ WhatsApp ගිණුම සම්බන්ධ කිරීමට **Session ID** එකක් අවශ්‍ය වේ. දන් බොට් පිහිටුවීමේදී ඔබ ලබාගත් Session ID එක `config.js` ගොනුවේ `SESSION_ID` පරාමිතියට ලබා දෙන්න.

#### 2. GitHub වෙත Upload කිරීම
1. GitHub ගිණුමට ලොග් වී නව **Repository** එකක් සාදන්න.
2. මෙම කේතය (All files) එම Repository එකට **Push/Upload** කරන්න.

#### 3. Heroku හෝ Koyeb වෙත Deploy කිරීම
බොට් එක 24/7 ක්‍රියාත්මක වීමට පහත සේවාවක් භාවිතා කරන්න:
*   **Heroku**: `app.json` ගොනුව හරහා පහසුවෙන් Deploy කළ හැක.
*   **Koyeb**: Docker හෝ Node.js runtime එක භාවිතයෙන් Deploy කරන්න.

### ⚙️ ප්‍රධාන විධානයන් (Main Commands)

*   `.menu` - සියලුම විධානයන් සිංහලෙන් බැලීමට.
*   `.ai [ප්‍රශ්නය]` - AI සමඟ සිංහලෙන් කතා කිරීමට.
*   `.alive` - බොට් පණ ගැන්වී ඇත්දැයි බැලීමට.

### 🛡️ වැදගත් (Important)
බොට් එක භාවිතා කිරීමේදී ඔබගේ WhatsApp ගිණුම අවහිර වීමේ අවදානමක් පවතින බැවින්, එය ඔබගේ වගකීමෙන් යුතුව භාවිතා කරන්න.
